
### Requirements
  * `pip install recbole`
  * `pip install causal-conv1d>=1.4.0`
  * `pip install mamba-ssm`
  
Our experimental platform is equipped with an NVIDIA V100 GPU, Ubuntu 22.04, PyTorch 2.1.2, and CUDA 12.1.
